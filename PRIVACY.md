# Política de Privacidade - PlenoPsi

A privacidade dos usuários é prioridade para o **PlenoPsi**. Este documento explica como tratamos os dados coletados.

---

## 1. Coleta de Dados
- Coletamos informações fornecidas pelo usuário (nome, e-mail, dados de pacientes e informações financeiras).  
- Dados de uso e acesso podem ser registrados para fins de melhoria do serviço.

---

## 2. Uso dos Dados
- Os dados são utilizados exclusivamente para fornecer e melhorar os serviços do PlenoPsi.  
- Não compartilhamos informações pessoais com terceiros sem consentimento, exceto quando exigido por lei.

---

## 3. Armazenamento e Segurança
- Os dados são armazenados em servidores seguros.  
- Adotamos medidas técnicas e organizacionais para proteger contra acessos não autorizados.

---

## 4. Direitos do Usuário
- O usuário pode solicitar acesso, correção ou exclusão de seus dados.  
- O tratamento segue as normas da **LGPD (Lei Geral de Proteção de Dados)**.

---

## 5. Alterações
- Esta política pode ser atualizada periodicamente.  
- Alterações relevantes serão comunicadas aos usuários.
